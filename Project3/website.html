Opleiding

De opleiding wordt georganiseerd door VDAB in samenwerking met Obelisk en MultiMedi met het doel de cursisten om te scholen tot junior .Net ontwikkelaars. Motivatie speelt hier dan ook een heel grote rol. Alle cursisten werden vooraf gescreend, getest en bevraagd naar hun motivatie. Uit het ruime aanbod van meer dan 100 kandidaten zijn de bovenstaande cursisten geselecteerd. Zij leggen dit traject af:

  • DEC 2020

    C#

    Basis syntax, structuren, principes en omgeving, programmatielogica, Visual Studio, OOP, Github, project

  • JAN - FEB 2021

    C# Advanced

    Generieken, lambda's, I/O, Multi-threading, SOLID & Design Principles (Robert C. Martin), WPF, project

  • FEB - MRT 2021

    Web

    XML, HTML, CSS, Bootstrap, intro Javascript, Ajax & JSON, presentatie, flyer, project: groeps- en persoonlijke websites

  • MRT - APR 2021

    Enterprise programmeren

    Building Automatisation, NuGet & Automated Building (Jenkins), Test Driven development & Unit Testing, TDD & Extreme Programming, NUnit, Gegevensanalyse, SQL, Entity Framework, WPF, project

  • MEI 2021

    Enterprise programmeren gevorderd

    Scrum & Agile, Javascript Framework, Typescript, Angular 8+, ASP.NET, MVC, Web API, Razor, project rond Web Application (MVC of REST)

  • JUN 2021

    .NET Core

    .NET Core: .NET CORE 3.1, EF, Nunit, REST, MVC, Blazor, Agile project
